一份优雅简约的简历

    优化构建，页面秒开无闪烁

    自适应屏幕兼容移动端

    支持部署到ghpages，可在线浏览

    自动生成 PDF，全自动化流程

使用

    fork本项目后再clone到本地修改

    进入项目目录执行npm i安装依赖

    执行npm run dev开始开发

    修改完后执行npm run pub发布到ghpages
